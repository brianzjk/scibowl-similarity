Science Bowl Similarity Browser

Static client-side review app for checking an uploaded MIT-format question sheet against the public corpus embedding bundle and against itself. Upload bundles are read from your machine in the browser. This page does not need a backend to run similarity search.

Generate Your Local Upload Bundle

Run the embedding model on your own machine, then load the resulting JSON bundle into this page. The bundle must be generated with the same embedding model and settings as the hosted corpus bundle.

This website expects `mixedbread-ai/mxbai-embed-large-v1` with the repo's default input format and `--include-answer` enabled. 
python -m pip install -e ".[embeddings]"
python -m scibowl.cli.main build-browser-upload-bundle `
  "YOUR_QUESTIONS.csv" `
  "your_upload_bundle.json" `
  --source-id your_upload `
  --tournament "Your Tournament" `
  --year 2026

The CSV should use the MIT columns documented in this repo. After the command finishes, load `your_upload_bundle.json` below.

Required CSV columns: `Type`, `Category`, `Format`, `Question`, `W`, `X`, `Y`, `Z`, `Answer`, `Accept`, `Do Not Accept`.

How This Page Works

1. Load the public corpus manifest, usually `./corpus/manifest.json` on the hosted site.

2. Pick your local upload bundle JSON from disk.

3. Click Analyze Upload to compute upload-vs-corpus and within-upload matches in your browser.

4. Filter by similarity and category, then review the pairs directly in the side-by-side viewer.

No CSV or embedding data is posted to a server by this page.
